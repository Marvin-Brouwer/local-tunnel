## Usage

Make sure you add the local domain to your hosts file.  
On windows: `C:\Windows\system32\drivers\etc\hosts`.  

```hosts
# localtunnel-server local development
127.0.0.1       localtunnel-dev
127.0.0.1       test.localtunnel-dev
```
